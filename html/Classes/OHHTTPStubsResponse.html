OHHTTPStubsResponse.m

Overview

Stubs Response. This describes a stubbed response to be returned by the URL Loading System, including its HTTP headers, body, statusCode and response time.

Adds convenience methods to manipulate JSON objects directly. Pass in an NSDictionary or NSArray to generate a corresponding JSON output.
